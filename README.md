

🧴 AI Skin Analyzer Ultra

An advanced AI-based skin analysis web application that uses computer vision techniques to detect skin type, analyze texture, estimate skin health, and generate actionable insights.

⸻

🚀 Features

* 📸 Upload face/skin image
* 👤 Face detection using OpenCV
* 🔴 Acne/spot detection (basic)
* 🧠 Skin type classification (Oily / Dry / Normal)
* 📊 Skin metrics:
    * Oil Level
    * Hydration Level
    * Skin Health Score
    * Confidence Score
* 📈 Visual progress indicators
* 🧾 Downloadable report (CSV)
* 🖥 Interactive UI with tabs

⸻

🛠️ Tech Stack

* Python
* Streamlit
* OpenCV
* NumPy
* Pandas
* Pillow

⸻

⚙️ How It Works

1. Upload a skin/face image
2. The system processes the image using computer vision
3. Extracts features like brightness and texture
4. Detects face and highlights regions
5. Generates:
    * Skin type
    * Health metrics
    * Acne estimation
    * Recommendations
